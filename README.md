# SelfTracker
A mod for Gorilla Tag that sends messages to a webhook whenever starting your game and joining/leaving rooms

---

# Installation

- Download the release from [here](https://github.com/iiDk-the-actual/SelfTracker/releases/latest)
- Drag the file `SelfTracker.dll` to your plugins folder
- Open Gorilla Tag once for files to initialize
- Input webhook URL into `SelfTracker/Configuration.Json`
- Edit other JSON files to customize messages
- Start Gorilla Tag if needed
